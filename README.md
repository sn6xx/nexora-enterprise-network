# Nexora Enterprise Network

## Overview

Nexora Enterprise Network is a large-scale enterprise networking project built using Cisco Packet Tracer.

The goal of this project is to design, implement, secure, and document a real-world enterprise network while covering CCNA technologies and enterprise networking best practices.

## Technologies

- Cisco Packet Tracer
- VLANs
- Inter-VLAN Routing
- OSPF
- EtherChannel
- STP
- ACL
- NAT/PAT
- DHCP
- DNS
- SSH
- IPv6
- Wireless LAN
- Network Security
